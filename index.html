Aprendiendo HTML5

El objetivo de esta página es explicar qué es HTML5, su estructura, sus etiquetas y su importancia en el desarrollo web.

Comenzar

¿Qué es HTML5?

HTML5 es la última versión del lenguaje HTML (HyperText Markup Language), un estándar utilizado para definir la estructura y el contenido de las páginas web. Este lenguaje permite organizar elementos como texto, imágenes, videos y otros recursos en una página web, facilitando su visualización en navegadores.

Estructura básica de HTML5

<!DOCTYPE html>
<html>
<head>
<title>Mi página</title>
</head>
<body>
</body>
</html>

Funcion doctype:

Esta instrucción es la primera línea que se escribe para codificar una página web en HTML. El doctype indica al navegador de qué tipo de documento se trata. Esta información permite al navegador utilizar sus conocimientos para aplicar las reglas propias del tipo indicado.

Funcion de html:

A partir de las cuales se cifra la apariencia y orden interno de una página web, así como los scripts o rutinas que operan dentro de ellas.

Función de head:

Define la información sobre una página web. Es el lugar donde se almacena la información que controla el aspecto y el comportamiento de tu sitio web.

Funcion de body:

Esta etiqueta es fundamental en HTML y se utiliza para definir el contenido principal de un documento en HTML.

Etiquetas principales

h1 - h6

Las etiquetas H1 a H6 en HTML son fundamentales para estructurar y jerarquizar el contenido de una página web. Cada etiqueta tiene un nivel de importancia diferente y define encabezados de distintos tamaños.

p

La etiqueta

se utiliza para definir párrafos de texto en HTML, proporcionando estructura, semántica y legibilidad al contenido de una página web.

img

La etiqueta en HTML se utiliza para insertar imágenes en una página web.

a

La etiqueta (a) en HTML se utiliza para crear enlaces a otras páginas web, archivos o ubicaciones dentro de la misma página.

button

La etiqueta button en HTML se utiliza para crear botones interactivos en una página web.

ul / ol / li

Las listas no ordenadas van dentro de la etiqueta

    Formularios en HTML5

    En HTML5, los formularios permiten recopilar datos del usuario y enviarlos a un servidor para su procesamiento. Se crean principalmente con la etiqueta

    y una serie de etiquetas y atributos que definen campos, botones y validaciones.

    Multimedia en HTML5

    La multimedia se maneja principalmente con las etiquetas img, audioy video, que permiten integrar imágenes, sonido y video directamente en una página web sin necesidad de plugins externos.

    Imágenes

    se utiliza para insertar imágenes en una página web. Es un elemento vacío (no tiene etiqueta de cierre) y requiere al menos el atributo src para funcionar.

    Audio

    se utiliza para insertar y reproducir audio directamente en una página web sin necesidad de plugins externos.

    Video

    se utiliza para incrustar contenido de video en una página web, como clips de películas o transmisiones de video.

    Accesibilidad

    proporciona un texto alternativo para las imágenes en caso de que no puedan visualizarse. Es esencial para mejorar la accesibilidad y el SEO de una página web.

    Conclusión

    Sin HTML, la web tal como la conocemos no existiría. Desde los encabezados hasta los párrafos, las imágenes y los enlaces, cada componente visual en una página web encuentra su origen en las etiquetas HTML. A través de una estructura jerárquica de elementos, este lenguaje permite organizar y presentar la información de manera coherente y accesible para los usuarios.
